# 前置准备

## npm 安装
推荐使用 npm 的方式安装，它能更好地和 webpack 打包工具配合使用。

::: code-group

```sh [npm]
$ npm install @myprint/design
```

```sh [pnpm]
$ pnpm add -D vitepress
```

```sh [yarn]
$ yarn add -D vitepress
```

```sh [yarn (pnp)]
$ yarn add -D vitepress vue
```

```sh [bun]
$ bun add -D vitepress
```

:::
