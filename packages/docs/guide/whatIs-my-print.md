配套服务

## 支持的控件

### 普通控件

| 名称   | 说明                    |
|------|-----------------------|
| 文本框  | 文本框                   |
| 时间控件 | 时间控件，打印时间             |
| 图片   | 支持选择本地图片、裁剪           |
| 数据表格 | 一个支持多级表头，表头排序、缩放的数据表格 |
| 横实线  | —                     |
| 竖实线  | —                     |
| 横虚线  | 虚线，支持修改虚线类型           |
| 竖虚线  | 虚线，支持修改虚线类型           |
| 页眉   | 页眉                    |
| 页脚   | 页脚                    |
| 页码   | 页码                    |

|---------|--------------|

| 手画板     | 手画板          |
